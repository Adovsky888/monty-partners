## 怎么运作

你会拿到专属链接 `monty.fast/r/YOUR_CODE` 和优惠码。凡是通过它们来并付费的客户，你都拿佣金 - **每笔付款的 30%，终身有效**，只要客户还在付费。大合作伙伴的比例更高。

到处都要带上你的链接或优惠码：没有它们，付款不会算到你头上。

最新条款和注册在 [monty.fast](https://monty.fast)。有问题：team@monty.fast。

## 素材使用

`promokit-assets/` 里的文件可以用来推广 Monty：帖子、快拍、短视频、封面。不可以：重画图标、改动标志比例、自造吉祥物、买 "Monty" 品牌词广告。
