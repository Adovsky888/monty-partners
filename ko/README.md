# Monty Partners

> # [🇷🇺 Русский](../ru/README.md) · [🇬🇧 English](../en/README.md) · 🇰🇷 한국어 · [🇨🇳 中文](../zh/README.md) · [🇪🇸 Español](../es/README.md)

Monty 파트너가 시작하는 데 필요한 모든 것: 바로 쓰는 카피, 브랜드 에셋, 완성된 영상.

**Monty**는 원본 영상을 완성된 릴스로 만들어 모든 채널에 알아서 올려주는 AI 에이전트입니다. [monty.fast](https://monty.fast)

## 작동 방식

개인 링크 `monty.fast/r/YOUR_CODE`와 쿠폰을 받습니다. 그 링크나 쿠폰으로 들어와 Monty를 결제한 고객마다 커미션이 발생합니다 - **결제 금액의 30%, 고객이 계속 결제하는 한 평생**. 규모가 큰 파트너는 요율이 더 올라갑니다.

어디에나 본인 링크나 코드를 넣으세요. 없으면 결제가 집계되지 않습니다.

최신 약관과 가입은 [monty.fast](https://monty.fast)에서. 문의: team@monty.fast.

## 에셋 사용

`assets/`의 파일은 Monty 홍보에 자유롭게 쓰세요: 게시물, 스토리, 릴스, 썸네일. 금지: 마크 다시 그리기, 로고 비율 변경, 자체 마스코트 제작, "Monty" 브랜드 키워드 광고 구매.
