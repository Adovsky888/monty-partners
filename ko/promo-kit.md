인플루언서 파트너를 위한 바로 쓰는 자료입니다. 복사하고, 내 코드만 넣고, 올리세요.

내 링크: `monty.fast/r/YOUR_CODE`
내 쿠폰: `YOUR_CODE` (시청자에게 시작 보너스를 줍니다)
내가 데려온 고객이 결제할 때마다 25%를 받습니다 - 그 고객이 결제하는 한 평생 (티어에 따라 최대 30%), 여기에 내가 초대한 파트너로부터 5%를 더 받습니다.

## 0. 이 키트 쓰는 법

1. 어디에든 내 추천 링크와 쿠폰을 꼭 넣으세요. 없으면 결제가 내 실적으로 잡히지 않습니다.
2. 아래 블록은 그대로 가져다 내 말투로 고쳐 쓰세요. 대본 읽듯 하지 마세요. 시청자는 대본 냄새를 바로 맡습니다.
3. 제일 잘 먹히는 건 과정을 카메라로 보여주는 것 (원본 클립 넣기 → Monty가 완성된 릴스를 뽑아냄). 말로만 설명하지 마세요.
4. 스토리 + 프로필 링크 고정이 일회성 게시물보다 훨씬 오래 굴러갑니다.

## 1. 포지셔닝 (한 줄)

내 채널 성격에 맞게 고르세요:

- "Monty는 내 촬영 원본을 알아서 완성된 릴스로 만들고 여기저기 다 올려주는 AI 에이전트예요."
- "Monty한테 아이디어만 말하면 편집된 릴스가 나와요. 손편집 없이."
- "소재만 올렸는데 Monty가 컷 따고, 자막 넣고, 음악 깔고, 업로드까지 끝냈어요. 끝."

이런 말은 피하세요: "영상 만드는 AI" (너무 흔함), "편집의 혁명" (알맹이 없음). 진짜 아픈 데를 찌르세요. 편집은 몇 시간을 잡아먹지만 Monty는 4분이면 끝납니다.

## 2. 릴스 / 쇼츠 대본 (30-45초)

### 대본 A - 고통 → 해결 (범용)

```
[0-3초] 카메라 보고 훅: "릴스 하나 편집하는 데 3시간 씁니다. 썼었죠."
[3-10초] "아이디어가 없는 게 문제가 아니에요. 앉아서 편집을 시작하는 게 문제죠."
[10-20초] 화면: 원본 영상을 Monty에 넣고 버튼 하나 누름.
[20-30초] 완성된 릴스 보여주기: 자막, 컷, 음악. 알아서 다 됐음.
[30-40초] "이거 4분 걸렸어요. 하룻밤이 아니라. 링크는 프로필에, 제 코드 쓰면 보너스."
```

### 대본 B - 챌린지 (다이내믹)

```
[0-3초] "챌린지. 완성된 릴스 하나를 4분 만에 뽑기. 갑니다."
[3-30초] 구석에 타이머. 실시간으로: 아이디어 → Monty → 완성된 릴스.
[30-40초] "성공. 예전엔 이게 제 저녁 전체였어요. 코드는 아래에, 직접 해보세요."
```

### 대본 C - 비포/애프터 (비주얼)

```
[0-5초] 왼쪽 "BEFORE": 쌓인 타임라인, 커피, 새벽. 오른쪽 "AFTER": 버튼 하나.
[5-25초] Monty가 대신 해주는 것들을 나레이션: 컷 편집, 자막, 사운드, 업로드.
[25-35초] 완성된 결과물 화면에. "링크랑 코드는 설명란에."
```

## 8. FAQ (시청자가 물어볼 것들)

- **유료인가요?** 무료로 시작할 수 있어요 (크레딧/바나나). 그 다음부터는 요금제입니다. 코드 넣으면 보너스 있고요.
- **편집 실력 있어야 하나요?** 아니요. 그게 핵심이에요. Monty가 대신 편집해줍니다.
- **어느 플랫폼에 올라가요?** 주요 SNS에 올라갑니다 (연동은 계속 늘어나는 중).
- **편집자를 완전히 대체하나요?** 노가다는 대체합니다. 감각이 들어가는 작가적 편집은 거들어주는 쪽이고, 방향은 본인이 잡습니다.
- **이거 광고예요?** 네, Monty랑 파트너 하고 있어요. 코드 쓰면 여러분은 보너스, 저는 수수료. 서로 좋은 거죠.

파트너십을 솔직하게 밝히면 신뢰가 올라갑니다. 숨기지 마세요.

## 9. 브랜드 규칙 (해도 되는 것 / 안 되는 것)

**해도 되는 것:**
- 실제 과정과 결과물 보여주기.
- 내 말투, 내 유머.
- 마스코트 원숭이 🐵, "Hey Monty".

**안 되는 것:**
- "100% 자동, 사람 필요 없음" 식의 약속. Monty는 어시스턴트고 창작자를 대신하지 않습니다.
- 시청자에게 수익이나 떡상을 보장하기.
- "평생 무료" 같은 낚시. 유료 요금제가 있습니다.
- 플랫폼 로고를 "공식 파트너십"처럼 쓰기.
- 본인 계정에 본인 코드 쓰기 (셀프 리퍼럴은 차단됩니다, monty.fast의 제휴 약관 페이지 참고).
