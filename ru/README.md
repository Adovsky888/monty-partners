## Как это работает

Ты получаешь персональную ссылку `monty.fast/r/ТВОЙ_КОД` и промокод. За каждого клиента, который пришёл по ним и оплатил Monty, идёт комиссия - **30% с каждого платежа, пожизненно**, пока клиент платит. У крупных партнёров ставка выше.

Везде ставь свою ссылку или код: без них оплата не засчитается.

Актуальные условия и регистрация - на [monty.fast](https://monty.fast). Вопросы: team@monty.fast.

## Использование ассетов

Файлы в `assets/` можно использовать для промо Monty: посты, сторис, ролики, превью. Нельзя: перерисовывать маркет, менять пропорции лого, делать своих маскотов, покупать брендовую рекламу на «Monty».
