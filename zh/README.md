## 怎么运作

你会拿到专属链接 `monty.fast/r/YOUR_CODE` 和优惠码。凡是通过它们来并付费的客户，你都拿佣金 - **每笔付款的 25%，终身有效**，只要客户还在付费。比例按等级最高升到 30%，此外你邀请来的合作伙伴，他们的每笔付款你还能额外拿 5%。

到处都要带上你的链接或优惠码：没有它们，付款不会算到你头上。

最新条款和注册在 [monty.fast](https://monty.fast)。有问题：team@monty.fast。

## 素材使用

`assets/` 里的文件可以用来推广 Monty：帖子、快拍、短视频、封面。不可以：重画图标、改动标志比例、自造吉祥物、买 "Monty" 品牌词广告。
